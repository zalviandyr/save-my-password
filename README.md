# Save My Password - Windows

## Feature

- CRUD account (SQLite)
- Backup restore (CSV)
- Filter

## Usage

- Clone this repository
- Open with VSCode
- Download all dependencies
  ```bash
  > flutter packages get
  ```
- Build
  ```bash
  > flutter build windows
  ```
- Execute application in folder
  `build\windows\runner\Release\save_my_password.exe`
